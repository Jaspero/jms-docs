# Autocomplete

## Interface

```typescript
interface AutocompleteData extends FieldData {
  dataSet: Array<{name: string; value: any}>;
  populate?: {
    collection: string;
    nameKey: string;
    valueKey?: string;
    orderBy?: string;
    filter?: WhereFilter;
    limit?: number;
  };
  autocomplete?: string;
  suffix?: HSD | string;
  prefix?: HSD | string;
}
```
