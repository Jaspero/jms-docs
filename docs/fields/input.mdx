# Input

## Interface

```typescript
interface InputData extends FieldData {
    /**
     * @default 'text'
     */
    type?: 'text' | 'number' | 'email' | 'color';

    /**
     * HTML autocomplete attribute
     * @default 'on'
     */
    autocomplete?: string;

    suffix?: HSD | string;
    prefix?: HSD | string;
}
```
